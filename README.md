# Quiz App

A simple Quiz App built using Python.

## Features
- Multiple quiz questions
- User input
- Score calculation
- Instant result

## Technologies Used
- Python 3

## How to Run

1. Download the project.
2. Open `main.py` in Python or PyDroid 3.
3. Run the program.
4. Answer the questions and see your score.
